Exemples d'integration des cartes

Appel du SDK

Chargement asynchrone du SDK javascript,
(une seule fois pour chaque page où doit être insérée une carte)

<script>(function(d, s, id) { var js, fjs = d.getElementsByTagName(s)[0]; if (d.getElementById(id)) return; js = d.createElement(s); js.id = id; js.async=true; js.src = "http://makinacorpus.github.io/elections/iframe-loader/mkc.js"; fjs.parentNode.insertBefore(js, fjs); }(document, 'script', 'mkc-iframe-loader'));</script>

OU

<script src="http://makinacorpus.github.io/elections/iframe-loader/mkc.js" id="mkc-iframe-loader" async></script>

Options d'intégration

Voici les différentes options disponibles. Pour les utiliser, il suffit d'ajouter les attributs correspondant sur la balise :

Choix de la Hauteur : data-height="pourcentage"

default: 75

La valeur représente la hauteur en pourcentage relative à la largeur de l'élément. Cela vous permet de définir un ratio de carte.

<div class="mkc-placeholder" data-source="2015-resultats-departementales" data-dpt="31 data-height="30"></div>

Affichage d'un logo : data-logo="url_du_le_logo"

default: undefined

Vous avez la possibilité de préciser un lien vers une image qui sera ajoutée dynamiquement au bas de la carte (par exemple, ajouter un logo)

<div class="mkc-placeholder" data-source="2015-resultats-departementales" data-dpt="31 data-logo="http://mydomain.com/mylogo.png"></div>

Empecher le zoom au scroll : data-zoomOnScroll="true/false"

default: true

Pour des soucis d'usabilité, vous pouvez désactiver la gestion du zoom par le scroll de la souris en passant l'attribut à false

<div class="mkc-placeholder" data-source="2015-resultats-departementales" data-dpt="31 data-zoomOnScroll="false"></div>
